### Real Time sudoku solver
##### This project will take sudoku picture as input from the webcam and run it thorugh  sudokusolver.py and finally render it out over the original picture.

#### How to use:
- download and run sukoku.py


##### libraries  used:
- OpenCV
- Keras
- Numpy
